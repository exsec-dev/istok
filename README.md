## Проект «Истокъ» - Frontend-часть

_ЛЦТ 2025. Задача "Сервис извлечения и индексирования информации из образов архивных документов (Ретроконверсия)"._

### Технологии

JavaScript/TypeScript

### Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск локально
npm start

# Запуск сборки
npm run build
```
